## 简要说明

Shelly 1PM Gen3 是一款紧凑型智能开关，集成电源测量功能，可通过智能手机、平板电脑、PC 或智能家居系统实现对电器的远程控制。该设备可在本地 Wi-Fi 网络中独立运行，也可通过基于云的智能家居服务进行管理。相比前代产品，其搭载了升级版处理器和更大内存，性能显著提升。

设备支持在任何具备互联网连接的地点进行远程访问、控制和监控，前提是设备始终连接至 Wi-Fi 路由器并保持在线。

专为安装于标准电气墙盒内、插座后方、开关后方或其他狭小空间而设计，可无缝融入现有布线系统。

内置网页界面，便于配置、实时监控和设备管理。

## 主要特性

- **Wi-Fi 连接：** 支持连接家庭 Wi-Fi 网络（802.11 b/g/n），实现远程监控与控制。
- **智能家居集成：** 兼容 Google Assistant、Amazon Alexa 和 Samsung SmartThings，支持语音控制与自动化。
- **带电量测量的智能开关：** 作为智能开关使用的同时，可实时监测所连接设备的功耗。
- **紧凑设计：** 小巧外形，适用于标准墙盒、插座或开关后方等狭小空间。
- **远程控制：** 可通过手机 App、网页浏览器或智能家居平台实现完全远程操作。
- **本地与云端运行：** 可独立运行于本地网络，也可接入云端服务。
- **性能增强：** 升级处理器与更大内存，响应更快，功能更强大。
- **远程访问：** 连接 Wi-Fi 后，可在任何有互联网的地方访问。
- **嵌入式网页界面：** 内置 Web UI，方便设置、监控与配置。
- **无线连接：** 支持 Wi-Fi（802.11 b/g/n）和蓝牙 4.2，具备明确的室内外覆盖范围。
- **BLE 网关：** 作为 Shelly BLU 设备与整个 Shelly 生态系统的桥梁——接收蓝牙信号，并转发至云端或本地设备。
- **物联网设备 Wi-Fi 范围扩展器：** 接收、放大并重新广播信号，扩大 Wi-Fi 覆盖范围。
- **脚本支持：** [Shelly 脚本语言功能](https://shelly-api-docs.shelly.cloud/gen2/Scripts/ShellyScriptLanguageFeatures/)
- **广泛集成选项：** 支持 HTTP API、MQTT(s)、Webhooks（HTTP/HTTPS）、UDP 及第三方智能家居系统。
- **KNX 支持：** 完全兼容 KNXnet/IP 通信协议。
- **定时调度：** 支持基于日期、时间、星期几、小时、分钟、秒的复杂自动化计划。
- **虚拟组件：** 可创建虚拟设备以实现高级自动化场景。[了解更多](https://shelly-api-docs.shelly.cloud/gen2/DynamicComponents/Virtual/)

## 应用场景

- **电器控制：** 远程管理与自动化灯光、风扇、加热器及其他电器设备。
- **电力监控：** 实时追踪连接电器的能耗，识别能效低下的设备。
- **智能家居集成：** 整合至自定义场景与智能生活流程中。
- **节能优化：** 通过识别高耗电设备，优化家庭能源使用。
- **远程监控：** 无论身处何地均可监督连接设备状态，确保安全与高效运行。

## 用户界面

### 输入

- 1 个（控制）按钮  
  - 长按 **5 秒**：启用设备热点与蓝牙配对模式。  
  - 长按 **10 秒**：执行恢复出厂设置。

### 输出

- 单色 LED 指示灯  
  - **热点启用，Wi-Fi 关闭**：1 秒开 / 1 秒关  
  - **Wi-Fi 启用，未连接**：1 秒开 / 3 秒关  
  - **已连接 Wi-Fi 网络**：持续亮起  
  - **云服务启用，未连接**：1 秒开 / 5 秒关  
  - **已连接 Shelly 云服务**：持续亮起  
  - **正在进行固件升级（OTA）**：0.5 秒开 / 0.5 秒关  
  - **长按 5 秒按钮**：0.5 秒开 / 0.5 秒关  
  - **长按 10 秒按钮**：0.25 秒开 / 0.25 秒关  

> *注：LED 状态按从上到下的优先级顺序生效，新状态会覆盖旧状态。*
