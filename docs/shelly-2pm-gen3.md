## 简要说明

Shelly 2PM Gen3 是一款紧凑型双通道智能开关，集成电源测量与遮盖控制功能。可通过智能手机、平板电脑、PC 或智能家居系统远程控制电器设备。该设备可在本地 Wi-Fi 网络上独立运行，或通过基于云的智能家居服务进行操作。

相比前代产品，本设备升级了处理器并增加了内存，显著提升了性能。此外，还支持电动百叶窗控制功能。

只要设备连接至 Wi-Fi 路由器并接入互联网，即可从任何有网络连接的位置实现远程访问、控制和监控。

专为安装在标准电气墙盒内、插座后方、开关后方或其他空间受限区域而设计。

设备内置网页界面，用于配置、监控和实时控制。

## 主要特性

- **带电量测量的智能开关：** 可控制设备的同时，实时测量所连电器的能耗。
- **紧凑设计：** 小巧外形，适合安装于标准墙盒、插座或开关后方等狭小空间。
- **远程控制：** 通过移动应用、平板电脑、PC 或智能家居平台实现完整远程操作。
- **本地与云端运行：** 支持本地网络独立运行，并可集成到基于云的自动化服务中。
- **改进的处理器与内存：** 升级硬件确保更快响应速度和更强的多任务处理能力。
- **远程访问：** 可从任意联网位置实现完全远程访问与监控。
- **嵌入式 Web 界面：** 内置网页用户界面，便于快速设置、监控与配置。
- **BLE 网关：** 作为蓝牙启用的 Shelly 设备与更广泛的 Shelly 生态系统之间的桥梁——接收 BLE 信号，并转发至云端或本地非 BLE 设备。
- **物联网设备 Wi-Fi 扩展器：** 通过接收、增强并重新传输信号，扩展 Wi-Fi 覆盖范围至更大区域。
- **脚本支持：** 支持使用 Shelly Script 实现自定义自动化（参见 [Shelly Script 语言功能](https://shelly-api-docs.shelly.cloud/gen2/Scripts/ShellyScriptLanguageFeatures/)）。
- **广泛集成支持：** 通过 HTTP API、MQTT(s)、Webhooks（HTTP/HTTPS）和 UDP 与第三方系统兼容。
- **KNX 支持：** 完全支持 KNXnet/IP 通信（仅限开关模式；需固件版本 1.5.0 或更高）。
- **定时调度：** 高级调度功能支持基于日期、时间、工作日、小时、分钟和秒的复杂操作。
- **虚拟组件：** 可创建动态虚拟组件以实现高级自动化逻辑（[了解更多](https://shelly-api-docs.shelly.cloud/gen2/DynamicComponents/Virtual/)）。
- **百叶窗模式：** 支持远程控制电动百叶窗、卷帘门、遮阳篷等设备。可测量连接电机的功耗。
  - 控制百叶窗/遮光帘位置
  - 调整百叶窗叶片倾斜角度  
  > ⚠️ **注意：** 仅可使用配备电子或机械限位开关的电机。连接设备前请确保限位已正确设置。

## 应用场景

- **电器控制：** 远程自动化控制灯光、风扇、加热器等电气设备。
- **电力监控：** 实时跟踪能耗，分析使用模式，提升能效。
- **智能家居集成：** 无缝融入智能家庭生态系统，用于场景创建与定时动作。
- **节能优化：** 识别高耗电设备并优化使用方式，降低电费支出。
- **远程监控与控制：** 无论身处何地，只要有网络连接即可管理已连接设备，即使不在家中也能操作。

## 设备电气接口

### 输入

- 两个接线端子上的开关/按钮输入：**S1** 和 **S2**
- 三个接线端子上的电源输入：**1 N (+)** 和 **2 L (Ʇ)**

### 输出

- 两个带电量测量的继电器输出，可通过接线端子访问

## 用户界面

### 输入

- 一个 **控制按钮**：
  - 长按 **5 秒**：进入 AP 模式并启用蓝牙配对。
  - 长按 **10 秒**：恢复出厂设置。

### 输出

- 单色 LED 指示灯：
  - **AP 已启用，Wi-Fi 未启用：** 1 秒 ON / 1 秒 OFF  
  - **Wi-Fi 已启用但未连接：** 1 秒 ON / 3 秒 OFF  
  - **已连接到 Wi-Fi 网络：** 持续亮起  
  - **已启用云服务但未连接：** 1 秒 ON / 5 秒 OFF  
  - **已连接至 Shelly Cloud：** 持续亮起  
  - **正在执行 OTA 更新：** 0.5 秒 ON / 0.5 秒 OFF  
  - **按下按钮 5 秒：** 0.5 秒 ON / 0.5 秒 OFF  
  - **按下按钮 10 秒：** 0.25 秒 ON / 0.25 秒 OFF  

> 🔁 *LED 状态按列出顺序优先级排序，高优先级状态会覆盖低优先级状态。*

## 故障排查

1. **检查电源供应**  
   - 检查电源线、插座是否正常工作，以及设备上的任何可见电源指示灯。

2. **检查接线连接**  
   - 确认所有导线均已牢固固定并正确插入端子。松动连接可能导致异常行为。

3. **复查设备设置**  
   - 在网页界面或 App 中再次确认配置项。确保设置符合您的使用需求。

4. **更新固件**  
   - 检查是否有可用的固件更新。保持设备更新可解决已知问题并提高稳定性。

5. **重启设备**  
   - 断电后等待 5–10 秒，再重新上电，以清除临时故障。

6. **检查网络配置**  
   - 确保输入的 Wi-Fi 凭据正确。测试网络连通性；如有需要，请重启路由器。

7. **检查物理状态**  
   - 查看是否存在损坏、过热或异常噪音迹象。

8. **确认兼容性**  
   - 验证与其他系统中设备（硬件/软件）的兼容性。

9. **监测环境条件**  
   - 避免超出规定范围的极端温度或湿度。

10. **评估电源质量**  
    - 电压波动或浪涌可能影响性能。建议使用浪涌保护器或稳压器。

> \*以上步骤为通用指南。特定问题可能需要额外诊断。若问题持续，请联系 [技术支持](https://support.shelly.cloud/en/support/home)。
