## 简要说明

Shelly 2PM Gen3 是一款紧凑型双通道智能开关，具备电力测量与遮盖控制功能。它可通过智能手机、平板电脑、PC 或智能家居系统远程控制电器设备。该设备可在本地 Wi-Fi 网络中独立运行，也可集成至基于云的智能家居服务。

相较于前代产品，Shelly 2PM Gen3 配备了升级版处理器和更大内存，性能显著提升。同时支持电动百叶窗控制功能。

只要设备连接到 Wi-Fi 路由器并接入互联网，即可从任何有网络连接的位置实现远程访问、控制和监控。

专为安装在标准电气墙盒内、插座后方、开关后方或其它狭小空间而设计，可无缝融入现有布线系统。

设备内置网页界面，用于配置、监控及实时调整设置。

## 主要特性

- **带电力测量的智能开关：** 控制连接设备的同时，实时测量功耗，提供用电情况洞察。
- **紧凑设计：** 小巧外形，适合安装于标准电气墙盒、插座或开关后方等狭小空间。
- **远程控制：** 可通过手机 App、平板电脑、PC 或集成智能家居平台进行完整远程操作。
- **本地与云端运行：** 可在本地网络独立工作，并可通过云服务进行管理。
- **升级处理器与内存：** 更强硬件带来更佳响应速度与可靠性。
- **远程访问：** 连接 Wi-Fi 后，无论身处何地均可通过互联网访问。
- **嵌入式 Web 界面：** 内置网页 UI，用于设备管理、状态监控与参数调整。
- **BLE 网关：** 将蓝牙设备（如 Shelly BLU）桥接到更广泛的 Shelly 生态系统——接收 BLE 信号并转发至云端或本地设备。
- **IoT 设备 Wi-Fi 范围扩展器：** 接收、放大并重新广播信号，扩大覆盖范围。
- **脚本支持：** 支持使用 Shelly Script 语言编写自定义自动化逻辑。  
  [了解更多 →](https://shelly-api-docs.shelly.cloud/gen2/Scripts/ShellyScriptLanguageFeatures/)
- **广泛集成支持：** 通过 HTTP API、MQTT(s)、Webhooks（HTTP/HTTPS）和 UDP 与第三方智能家居系统兼容。
- **KNX 支持：** 使用 KNXnet/IP 协议接入 KNX 网络（仅支持开关模式；需固件版本 1.5.0 或更高）。
- **定时任务：** 可创建复杂的基于时间的自动化规则，精确控制日期、时间、星期、小时、分钟和秒。
- **虚拟组件：** 使用动态虚拟组件实现高级自动化逻辑。  
  [了解更多 →](https://shelly-api-docs.shelly.cloud/gen2/DynamicComponents/Virtual/)
- **百叶窗模式：** 支持远程控制电动百叶窗、卷帘门、遮阳篷等设备。可测量连接电机的功耗。  
  - 推荐使用：配备电子或机械限位开关的电机。  
  - 连接设备前必须正确设置电机限位。  
  - 控制功能包括：  
    - 百叶窗、卷帘、遮光板的位置  
    - 百叶窗叶片的倾斜角度

## 应用场景

- **电器控制：** 远程自动化控制灯光、风扇、加热器等电气设备。
- **电力监测：** 实时追踪能耗，识别低效设备并减少用电。
- **智能家居集成：** 嵌入到您的智能家庭系统中的场景、流程和自动化中。
- **能效优化：** 利用电力数据识别高耗电设备，优化使用习惯。
- **远程监控与控制：** 即使不在家中，也能通过联网方式管理设备。

## 用户界面

### 输入

- 一个（控制）按钮：
  - 长按 **5 秒**：进入热点模式并启用蓝牙配对
  - 长按 **10 秒**：恢复出厂设置

### 输出

- 单色 LED 指示灯：
  - **AP 已启用，Wi-Fi 未启用：** 1 秒亮 / 1 秒灭
  - **Wi-Fi 已启用但未连接：** 1 秒亮 / 3 秒灭
  - **已连接至 Wi-Fi 网络：** 持续常亮
  - **已启用云服务但未连接：** 1 秒亮 / 5 秒灭
  - **已连接至 Shelly 云：** 持续常亮
  - **OTA（空中升级）：** 0.5 秒亮 / 0.5 秒灭
  - **长按 5 秒按钮：** 0.5 秒亮 / 0.5 秒灭
  - **长按 10 秒按钮：** 0.25 秒亮 / 0.25 秒灭

> 🔁 **注意：** 上述 LED 状态按从上到下的优先级执行，后续状态将覆盖先前状态。

## 故障排除

1. **确保供电正常：**
   - 检查电源线、插座及设备上的任何可见指示灯。

2. **检查所有连接：**
   - 确认接线、端子和连接器牢固且安装到位。松动连接可能导致间歇性故障。

3. **检查设备设置：**
   - 确保配置（如 Wi-Fi 凭据、定时任务、自动化规则）符合预期行为。

4. **更新固件/软件：**
   - 检查是否有可用更新。新固件通常可修复漏洞并提高稳定性。

5. **重启设备：**
   - 断电，等待 10 秒后重新上电。可清除临时故障。

6. **验证网络连接：**
   - 确保 Wi-Fi 设置正确。如有需要，重启路由器或接入点。

7. **检查物理状态：**
   - 查看是否有损坏、过热迹象或异常行为（如烧焦味、LED 闪烁）。

8. **确认兼容性：**
   - 确保设备与其他系统组件（软硬件）兼容。

9. **监控环境条件：**
   - 避免超出规定范围的极端温度或湿度。

10. **评估电能质量：**
    - 电压波动或浪涌可能影响性能。建议使用浪涌保护器或稳压器。

> 📌 *以上步骤为通用指导。特定问题可能需要定制解决方案。若问题持续存在，请联系 [Shelly 技术支持](https://support.shelly.cloud/en/support/home)。*
