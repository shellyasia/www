## 简要说明

Shelly 2PM Gen4 是一款紧凑型双通道智能开关，具备功率测量和盖板控制功能。它支持通过智能手机、平板电脑、PC 或家庭自动化系统远程控制电器设备。该设备可在本地 Wi-Fi 网络中独立运行，也可集成到基于云的家庭自动化服务中。

相较于前代产品（Shelly 2PM Gen3），Gen4 版本配备了升级的处理器、更大的内存以及更强大的连接能力。它支持百叶窗控制功能，与 Gen3 型号类似。

只要设备连接至 Wi-Fi 路由器并接入互联网，即可实现从任何有网络连接的位置进行远程访问、控制和监控。

设计用于安装在标准电气墙盒内，或隐藏于电源插座、灯控开关及其他狭小空间后方。

设备内置网页界面，用于配置、监控和控制。其采用多协议无线 MCU，支持 Zigbee 和蓝牙，实现安全灵活的集成。

本设备兼容 Matter（默认设备配置文件：Switch）。

## 主要特性

- **带功率测量的智能开关：** 远程控制设备的同时，实时测量功耗，实现能源监控。
- **紧凑设计：** 小巧外形，非常适合安装在标准墙盒内、插座后方或空间受限区域。
- **远程控制：** 通过移动应用、网页浏览器或家庭自动化平台，从任意位置操作电器。
- **本地与云端运行：** 可在本地网络独立工作，也可连接云端服务以实现远程访问。
- **性能提升与 Zigbee 支持：** 配备增强型处理器和内置 Zigbee 连接，扩展生态系统集成能力。
- **Zigbee 连接：** 支持设置时设备入网，并可构建网状网络。
- **远程访问：** 连接 Wi-Fi 后，可通过互联网实现全面远程监控与控制。
- **嵌入式 Web 界面：** 内置界面，便于配置、查看状态及调整参数。
- **BLE 网关：** 将蓝牙 enabled 的 Shelly 设备桥接到更广泛的 Shelly 生态系统——接收 BLE 信号并转发至云端或本地设备。
- **Zigbee 范围扩展器：** 通过中继信号，延长 Zigbee 网络覆盖范围。
- **Wi-Fi 范围扩展器：** 接收、放大并重新传输信号，扩大 Wi-Fi 覆盖范围。
- **脚本支持：** 支持通过 Shelly Script 实现自定义自动化（参见 [Shelly Script 语言特性](https://shelly-api-docs.shelly.cloud/gen2/Scripts/ShellyScriptLanguageFeatures/)）。
- **广泛集成选项：** 通过 HTTP API、MQTT(s)、Webhooks（HTTP/HTTPS）和 UDP 与第三方系统兼容。
- **KNX 支持：** 支持通过 KNXnet/IP 通信（参见 [KNX 文档](https://kb.shelly.cloud/knowledge-base/shelly-knx-documentation#:~:text=This%20documentation%20applies%20to%20Shelly%20devices%20with%20built-in%20KNXnet/IP%20support) 和 [集成指南](https://shelly-api-docs.shelly.cloud/gen2/Integrations/KNX/)）。
- **定时任务：** 支持基于日期、时间、星期、小时、分钟和秒创建复杂的时间驱动自动化。
- **虚拟组件：** 使用动态虚拟组件实现高级逻辑与自动化（参见 [虚拟组件指南](https://shelly-api-docs.shelly.cloud/gen2/DynamicComponents/Virtual/)）。
- **百叶窗模式：** 控制电动百叶窗、卷帘门、遮阳篷等设备。可测量电机功耗。需使用带有电子或机械限位开关的电机。连接前请确保限位已正确设置。

## 应用场景

### 电器控制
远程管理并自动化照明、风扇、加热器及其他电气设备。

### 功率监控
实时跟踪连接设备的能耗，识别低效设备，促进节能。

### 智能家居自动化
集成至智能家居生态系统，创建场景、定时任务和自动化流程。

### 能效优化
利用功率测量数据检测高耗电设备，优化使用模式。

### 远程监控
无论身处何地，只要有互联网连接，即可远程监控和控制设备。

---

### 智能家居应用场景

#### 两个电路独立控制
- 通过一个设备控制两个独立的照明回路或电器。
- 适用于双开关布局（如客厅与走廊）。
- 可独立监测每个通道的功耗。

#### 智能百叶窗 / 遮帘控制
- 自动化控制电动百叶窗、卷帘门或窗帘。
- 支持校准，实现精确的开/关/停定位。
- 支持百叶窗叶片倾斜控制。
- 可根据时间、太阳高度角或天气条件设定定时任务。

#### 电器能耗监控
- 同时监控两个设备的用电情况（如咖啡机 + 烤面包机）。
- 当功率超过阈值时触发警报或执行动作。

#### 浴室与厨房通风
- **通道 1：** 通过湿度触发自动化控制浴室排气扇。
- **通道 2：** 通过定时任务或温感器控制镜面加热器或毛巾架。

---

### 商业 / 办公场景

#### 会议室或区域照明
- 从单一设备自动化控制相邻房间或区域的照明。
- 可追踪各通道单独功耗，用于能源审计。

#### 展示或广告牌控制
- **通道 1：** 控制商店招牌灯光。
- **通道 2：** 控制促销显示屏电视或自助终端。
- 支持定时运行或运动触发激活。

#### 窗帘遮蔽自动化
- 自动化控制会议室或大堂的百叶窗或遮光帘。
- 可与环境光传感器或日历中的会议安排联动。

---

### 工业与设施应用场景

#### 双电机控制
- 控制并监控两个低负载电机（如风机、阀门、水泵）。
- 可与环境传感器（温度、水位）集成，实现自动响应。

#### 分路负载监控
- 监测同一电路上两个不同负载的能耗。
- 适用于关键或备用设备（路由器、调制解调器、UPS 系统）的运行跟踪。

## 用户界面

### 输入
- **一个（控制）按钮**
  - 长按 5 秒：启用设备热点并开启蓝牙连接。
  - 长按 10 秒：恢复出厂设置。
  - 快速连按 5 次：从 Matter 固件（默认）切换至 Zigbee 模式。
  - 快速连按 3 次：进入 Zigbee 加入模式（持续 2 分钟；设备将在家庭自动化平台中通过 Zigbee 网关显示）。

### 输出
- **LED（单色指示灯）**
  - AP 已启用，Wi-Fi 未启用：1 秒亮 / 1 秒灭  
  - Wi-Fi 已启用但未连接：1 秒亮 / 3 秒灭  
  - 已连接 Wi-Fi 网络：持续亮  
  - 已启用云服务但未连接：1 秒亮 / 5 秒灭  
  - 已连接 Shelly 云：持续亮  
  - 正在进行固件升级（OTA）：0.5 秒亮 / 0.5 秒灭  
  - 按钮长按 5 秒：0.5 秒亮 / 0.5 秒灭  
  - 按钮长按 10 秒：0.25 秒亮 / 0.25 秒灭  

> *注：状态按优先级顺序处理，后续状态会覆盖先前状态。*
