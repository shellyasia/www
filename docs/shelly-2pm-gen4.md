## 简要描述

Shelly 2PM Gen4 是一款紧凑型双通道智能开关，集成电源测量与遮盖控制功能。可通过智能手机、平板电脑、PC 或家庭自动化系统远程控制电器。该设备可在本地 Wi-Fi 网络上独立运行，也可通过基于云的家庭自动化服务进行操作。

相比前代产品（Shelly 2PM Gen3），本设备升级了处理器并增加了内存容量，并支持与前代相似的百叶窗功能。

只要设备连接到 Wi-Fi 路由器并接入互联网，即可从任何有网络连接的位置实现远程访问、控制和监控。

专为安装在标准电气墙盒内、插座后方、开关后方或其他空间受限位置而设计。

设备内置网页界面，用于配置、监控和控制。同时支持通过 Zigbee 和蓝牙的多协议无线通信，确保安全且灵活的集成。

此设备兼容 Matter（默认设备配置文件：Switch）。

## 主要特性

- **带电量测量的智能开关：** 作为智能开关使用，可实时监测连接电器的功耗。
- **紧凑设计：** 小巧外形，适合安装于标准墙盒、插座后方、开关后方或狭小空间。
- **远程控制：** 可通过移动应用、平板电脑、PC 或家庭自动化平台远程控制设备。
- **本地与云端运行：** 可独立运行于本地网络，也可集成至基于云的自动化服务。
- **改进的处理器与 Zigbee 连接：** 内置 Zigbee 支持，性能提升，扩展生态系统集成能力。
- **Zigbee 支持：** 可加入 Zigbee 网络，在初始设置和网络扩展中非常有用。
- **远程访问：** 无论身处何地，只要有互联网连接，即可实现完全远程访问与监控。
- **嵌入式 Web 界面：** 内建界面，用于配置、状态监控和参数调整。
- **BLE 网关：** 通过将蓝牙设备信号转发至云端或本地设备，连接蓝牙设备至更广泛的 Shelly 生态系统。
- **Zigbee 范围扩展器：** 接收、增强并重新广播信号，有效扩展 Zigbee 网络覆盖范围。
- **Wi-Fi 范围扩展器：** 接收、放大并重发信号，扩大 Wi-Fi 覆盖范围。
- **脚本支持：** 支持通过 Shelly Script 实现自定义自动化（参见 [Shelly Script 语言功能](https://shelly-api-docs.shelly.cloud/gen2/Scripts/ShellyScriptLanguageFeatures/)）。
- **广泛集成支持：** 通过 HTTP API、MQTT(s)、Webhooks（HTTP/HTTPS）和 UDP 与第三方家庭系统兼容。
- **KNX 集成：** 支持 KNXnet/IP 通信（[文档](https://kb.shelly.cloud/knowledge-base/shelly-knx-documentation#:~:text=This%20documentation%20applies%20to%20Shelly%20devices%20with%20built-in%20KNXnet/IP%20support)）——[API 参考](https://shelly-api-docs.shelly.cloud/gen2/Integrations/KNX/)。
- **定时调度：** 基于日期、时间、星期几、小时、分钟和秒创建复杂调度。
- **虚拟组件：** 使用虚拟组件实现高级自动化逻辑 —— [了解更多](https://shelly-api-docs.shelly.cloud/gen2/DynamicComponents/Virtual/)。
- **百叶窗模式：** 支持电动百叶窗、卷帘门、遮阳篷和窗帘。可测量连接电机的功耗。需配备电子或机械限位开关；使用前请确保正确配置。

## 应用场景

### 电器控制
远程管理并自动化灯光、风扇、加热器及其他电器设备。

### 用电量监测
实时监控连接电器的能耗，识别低效设备，促进节能。

### 智能家居自动化
集成至智能家居生态系统，创建个性化场景、自动化流程和日常习惯。

### 能效优化
利用用电数据检测高耗电设备，优化使用模式。

### 远程监控
无论身在何处，只要有网络，即可查看和控制设备——非常适合远程管理。

---

### 智能家居应用场景

#### 两个电路的独立控制
- 通过一个设备控制两个独立的照明回路或电器。
- 适用于双开关布局（如客厅与走廊灯）。
- 可独立监控每个通道的功率消耗。

#### 智能百叶窗/遮帘控制
- 自动化控制电动百叶窗、卷帘或窗帘。
- 支持校准以实现精确位置控制（开/关/停）。
- 百叶窗叶片角度调节（倾斜控制）。
- 基于时间、太阳高度角或天气状况自动运行。

#### 电器能耗监控
- 监测两个不同电器的用电情况（如咖啡机 + 烤面包机）。
- 当功率阈值被超过时触发警报或执行动作。

#### 浴室与厨房通风
- **通道 1：** 通过湿度触发自动控制浴室排风扇。
- **通道 2：** 通过定时或温度触发控制镜面加热器或毛巾架。

---

### 商业 / 办公场所应用场景

#### 会议室或区域照明
- 通过单个设备自动化控制两个相邻区域的照明。
- 可追踪各区域单独的用电量，用于能源审计。

#### 展示或广告牌控制
- **通道 1：** 商店招牌照明。
- **通道 2：** 促销展示电视或数字信息亭。
- 可设定运行时间表，或通过运动检测触发。

#### 窗帘遮蔽自动化
- 自动化会议厅或大堂的百叶窗或遮光帘。
- 可联动环境光传感器或日历中的会议安排。

---

### 工业与设施应用场景

#### 双电机控制
- 控制并监控两个低负载电机（如风机、阀门、水泵）。
- 可与环境传感器（温度、水位等）联动，实现自动化响应。

#### 分负荷监控
- 监测共享电路上两个独立负载的能耗。
- 适用于跟踪关键或备用设备（如路由器、调制解调器等）。

## 用户界面

### 输入
- 一个 **控制按钮**：
  - 长按 **5 秒**：启用设备热点并进入蓝牙配对模式。
  - 长按 **10 秒**：恢复出厂设置。
  - 快速连续按下 **5 次**：从 Matter 固件切换至 Zigbee 模式。
  - 快速连续按下 **3 次**：进入 Zigbee 加入模式（持续 2 分钟）。设备将在您的家庭自动化平台中通过 Zigbee 网关出现。

### 输出
- 单色 LED 指示灯：
  - **AP 已启用，Wi-Fi 未连接**：1 秒亮 / 1 秒灭  
  - **Wi-Fi 已启用但未连接**：1 秒亮 / 3 秒灭  
  - **已连接到 Wi-Fi 网络**：常亮  
  - **已启用云服务但未连接**：1 秒亮 / 5 秒灭  
  - **已连接至 Shelly Cloud**：常亮  
  - **正在进行 OTA 更新**：0.5 秒亮 / 0.5 秒灭  
  - **长按按钮 5 秒**：0.5 秒亮 / 0.5 秒灭  
  - **长按按钮 10 秒**：0.25 秒亮 / 0.25 秒灭  

*注：状态按从上到下的优先级顺序执行，高优先级状态会覆盖低优先级状态。*
